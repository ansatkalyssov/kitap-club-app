# Кітап Клубы — мобильді қосымша

Expo (React Native) қосымшасы. Веб-нұсқамен бірдей Supabase backend-ін қолданады.

Бастапқы нұсқаға кіретіні:

- Кіру / тіркелу (email + пароль)
- Күнделікті оқу жоспары: уақыт немесе бет саны бойынша мақсат, таймер/бет журналы, серия (streak), соңғы 7 күн тарихы
- Мақсатты орындамасаңыз күнделікті жергілікті хабарландыру (push notification)
- Кітап трекері: жеке трекерлер тізімі (клуб бойынша топталған), жаңа трекер жасау, прогресті күнделікті тіркеу, оқу тарихы
- Клубтар: барлық белсенді клубтарды шолу/іздеу, клубқа қосылу/шығу (макс 3 клуб), клуб мәліметі (кітап жоспары, кездесулер)
- Пікір алмасу: тректерді оқу және жауап жазу (клуб мүшелері мен жүргізушілер үшін)
- Жүргізуші функциялары: клуб жасау (CreateClubForm), клубқа жаңа кітап жоспарын қосу, жаңа талқы тақырыбын ашу (CreateAnalysisForm), оқырмандар үлгерімін көру

> Жоспарға мүшелер қосылғанда трекерлер автоматты жасалады (JoinClubButton арқылы), бірақ жүргізуші жаңа жоспар қосқанда барлық қолданыстағы мүшелерге трекерлерді жаппай жасау веб-нұсқадағыдай admin/service-role арқылы жүзеге аспайды (қауіпсіздік себебінен мобильде сервис кілті болмауы керек) — әр оқырман өз трекерін "Кітап трекері" бөлімінен өзі қосады.

## Алғашқы баптау

1. Тәуелділіктерді орнатыңыз:

   ```bash
   npm install
   ```

2. `.env.example` файлын `.env` деп көшіріп, Supabase мәндерін енгізіңіз (веб қосымшаның `.env.local` файлындағы `NEXT_PUBLIC_SUPABASE_URL` / `NEXT_PUBLIC_SUPABASE_ANON_KEY` мәндерімен бірдей):

   ```bash
   EXPO_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
   EXPO_PUBLIC_SUPABASE_ANON_KEY=sb_publishable_xxxxx
   ```

   > Тек public/anon кілтті қолданыңыз. `SUPABASE_SERVICE_ROLE_KEY` мобильді қосымшада ешқашан қолданылмауы керек.

## Іске қосу

```bash
npx expo start
```

- Телефонда тексеру үшін **Expo Go** қосымшасын орнатып, терминалдағы QR кодты сканерлеңіз (телефон мен компьютер бір Wi-Fi желісінде болуы керек).
- Android эмулятор: терминалда `a` басыңыз.
- iOS симулятор (тек macOS): терминалда `i` басыңыз.

## Хабарландырулар туралы

Күнделікті еске салғыш — құрылғыда жоспарланатын **жергілікті** хабарландыру (`expo-notifications`). Ол "Жоспарды сақтау" батырмасын басқанда, "Еске салғыш" қосулы болса, `reading_goals.reminder_time` уақытына орнатылады. Бірінші рет хабарландыруға рұқсат сұралады.

> Expo Go-да Android push токендері шектеулі болуы мүмкін, бірақ жергілікті (local) хабарландырулар Expo Go-да да жұмыс істейді.

## Келесі қадамдар
